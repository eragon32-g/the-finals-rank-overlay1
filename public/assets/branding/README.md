# Branding — Nexaris

Cartella per i loghi e gli elementi di marca della piattaforma **Nexaris**.

## Asset previsti

| File | Uso |
|------|-----|
| `nexaris-logo.png` | Logo completo (icona + wordmark) |
| `nexaris-symbol.png` | Solo simbolo / mark, senza testo |
| `favicon.png` | Icona scheda browser (quadrata) |
| `logo-dark.png` | Variante per sfondi chiari |
| `logo-light.png` | Variante per sfondi scuri |

## Note

- Sfondo trasparente (PNG) consigliato.
- Mantenere margini di sicurezza intorno al logo.
- Esportare a risoluzione alta (es. 2x) per schermi retina.
- I file di lavoro / bozze vanno in `placeholders/`.
