# Overlays — Nexaris

**Template** e **preview** degli overlay, divisi per gioco.

## Struttura

- `the-finals/`
- `valorant/`
- `apex/`

Ogni gioco contiene le immagini di anteprima dei vari tipi di overlay.

## Anteprime previste per ogni gioco

| File | Tipo overlay |
|------|--------------|
| `compact-preview.webp` | Overlay compatto |
| `creator-preview.webp` | Overlay creator / social |
| `team-preview.webp` | Overlay team / clan |
| `ranked-preview.webp` | Overlay ranked |

## Note

- Le preview servono per la galleria/scelta overlay nell'interfaccia.
- Formato `.webp` consigliato, proporzioni coerenti tra i tipi.
