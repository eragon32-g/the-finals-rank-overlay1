# UI — Nexaris

Elementi grafici **generici** dell'interfaccia, riutilizzabili in tutte le pagine.

## Sottocartelle

| Cartella | Contenuto |
|----------|-----------|
| `icons/` | Icone menu, azioni, social (PNG/SVG) |
| `buttons/` | Stati e varianti dei bottoni |
| `effects/` | Glow, bagliori, gradient, particelle |
| `panels/` | Pannelli, card, cornici, sfondi sezione |

## Note

- Preferire SVG per icone scalabili, PNG/WEBP per effetti.
- Sfondo trasparente dove possibile.
- Stile coerente dark gaming/esports.
