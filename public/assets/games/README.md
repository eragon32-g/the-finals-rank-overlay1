# Games — Nexaris

Ogni gioco supportato ha una **cartella dedicata** con i propri asset (icona, banner, sfondo).

## Giochi attuali

- `the-finals/`
- `valorant/`
- `apex/`

## Asset previsti per ogni gioco

| File | Uso |
|------|-----|
| `icon.png` | Icona quadrata del gioco (liste, menu) |
| `banner.webp` | Banner orizzontale (card, header) |
| `background.webp` | Sfondo dedicato alla sezione del gioco |

## Note

- Per aggiungere un nuovo gioco: crea una nuova sottocartella con lo stesso set di file.
- Mantenere nomi file identici tra i giochi per semplificare il codice.
