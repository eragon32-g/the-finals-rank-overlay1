# Backgrounds — Nexaris

Sfondi a tutta pagina usati dalle schermate principali della piattaforma.

## Asset previsti

| File | Schermata |
|------|-----------|
| `home-bg.webp` | Landing / Home |
| `login-bg.webp` | Pagina di accesso |
| `dashboard-bg.webp` | Dashboard utente |
| `studio-bg.webp` | Studio / editor overlay |

## Specifiche consigliate

- Risoluzione: **1920×1080**
- Formato: **.webp** (preferito)
- Stile: **dark gaming / esports**
- **Nessun testo** dentro l'immagine (il testo è gestito dall'interfaccia)

## Note

- Evitare zone troppo chiare al centro: l'UI ci va sopra.
- File di lavoro / bozze in `placeholders/`.
