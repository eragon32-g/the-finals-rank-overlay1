# THE FINALS Rank Overlay V7 - All Rank Badges

Questa versione risolve il problema dello stemma che resta sul livello 1.

## Come funziona

Ora puoi scegliere lo stemma preciso tramite `badgeFile`.

Esempi:

```text
badgeFile=platinum-4
badgeFile=platinum-3
badgeFile=platinum-2
badgeFile=platinum-1
```

Lista completa:
- bronze-4, bronze-3, bronze-2, bronze-1
- silver-4, silver-3, silver-2, silver-1
- gold-4, gold-3, gold-2, gold-1
- platinum-4, platinum-3, platinum-2, platinum-1
- diamond-4, diamond-3, diamond-2, diamond-1
- ruby

## Link manuale esempio

```text
https://TUO-PROGETTO.vercel.app/?mode=manual&player=erdragon32%232577&league=Platinum&division=II&rankScore=37705&badgeFile=platinum-2
```

Apri:

```text

```

e scegli direttamente il rank preciso.

https://id.embark.games/images/leaderboards/leagues/
```

public/assets/badges/
```
