## Branding

Puoi usare:
- `brandText=VOIDRAGE32`
- `discordText=DISCORD.GG/TUOLINK`
- `extraText=JOIN%20THE%20VOID`

Se `extraText` è vuoto, l'overlay usa automaticamente:
`brandText • discordText`
